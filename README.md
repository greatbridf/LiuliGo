# LiuliGo
[![Build Status](https://travis-ci.com/greatbridf/LiuliGo.svg?token=ncMN4paT5YpmskkyPYRb&branch=master)](https://travis-ci.com/greatbridf/LiuliGo)  
A simple go-based hacg.me articles getter
## Build args
### Archs
```shell
make os=[OS] arch=[ARCH]
```
OS:
- Windows: `windows`
- Linux: `linux`
- macOS: `darwin`
---
ARCH:
- x86: `386`
- x64: `amd64`
- ARM: `arm`
